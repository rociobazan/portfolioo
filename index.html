Pokedex Web

Funcionalidad:

Pókedex es una aplicación web desarrollada con C#, .NET Framework, HTML, CSS y Bootstrap, con conexión a base de datos SQL. Nos presenta los pókemons en formato de tarjetas, como así también en grilla. Permite modificar un pokemon que ya tenemos, precargando un formulario con los datos del mismo, para que sólo escribamos el dato que elegimos editar. También brinda la posibilidad de agregar un nuevo bichito con sus respectivos datos a nuestra lista, presentando en ese caso un formulario vacío a completar.

Pokedex de escritorio

Funcionalidad:

Desarrollada en C# .Net framework, Pokedex es una aplicación de escritorio construída con arquitectura en capas, POO y conexiones a Base de Datos. Permite el registro de los famosos Pokemons, agregar nuevos personajes incorporando sus datos, y una imagen que puede ser asignada vía url o subida localmente. Permite también modificar, eliminar y filtrar pokemons por filtro rápido o criterios más específicos.
